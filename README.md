Overview: A website where people can request for blood and donate blood. Donor Volunteer and Admin has different role.

Live link: https://imaginative-mochi-0494cf.netlify.app/

Technologies: React Router, React.js, Tailwind, DaisyUI, Firebase.

Core features: Request for blood, Donate blood, Manage bonation status, Manage users role, Delete reauest, login, register (email+password).
